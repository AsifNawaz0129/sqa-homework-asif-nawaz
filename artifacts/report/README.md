## Manually Trigger Playwright Tests

You can manually trigger the Playwright test suite via GitHub Actions if you need to run the tests outside of a push to the `main` branch.

### How to Trigger:

1.  Go to the workflow dispatch page: [Trigger Workflow](https://github.com/AsifNawaz0129/sqa-homework-asif-nawaz/actions/workflows/playwright.yml)
2.  On the right sidebar, click on "Run workflow".
3.  Ensure the branch is set to `main` 
4.  Click the green "Run workflow" button.

This will start a new workflow run, and once completed, the updated report will be deployed to the [Live Report](https://asifnawaz0129.github.io/sqa-homework-asif-nawaz/) link.
